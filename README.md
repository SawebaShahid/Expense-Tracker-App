# 💰 Expense Tracker Application

A full-stack **Expense Tracker** built to help users manage their personal finances. The application allows users to record income and expense transactions, automatically calculates the current balance, stores data securely in MongoDB, and provides a clean, responsive dashboard for tracking financial activity.

---

## ✨ Features

- ➕ Add income transactions
- ➖ Add expense transactions
- 💰 Automatic balance calculation
- 📜 Transaction history
- 💾 Store data in MongoDB
- 📊 Responsive dashboard interface
- 📱 Mobile-friendly design

---

## 🛠️ Tech Stack

### Frontend
- ⚛️ React.js
- 🎨 CSS / Tailwind CSS
- 🌐 Axios

### Backend
- 🟢 Node.js
- 🚀 Express.js
- 🍃 MongoDB
- 🔗 Mongoose

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/your-username/expense-tracker-app.git
```

### Install dependencies

#### Frontend

```bash
cd client
npm install
```

#### Backend

```bash
cd server
npm install
```

### Run the application

#### Start the backend

```bash
npm run dev
```

#### Start the frontend

```bash
npm start
```
